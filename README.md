## Repository policy
- 記事は `public/` 配下の Markdown を管理対象とします。
- `public/` には「Qiitaに投稿したい Markdown だけ」を置きます（投稿対象外の生成物は置かない）。
- Qiita CLI が生成する可能性のある `.remote/` は **リポジトリ直下**で管理し、Gitには含めません（`.gitignore`）。
