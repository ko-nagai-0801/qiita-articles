**1カラム自己紹介ページに「作品紹介」を追加して、ポートフォリオらしさを出します。**  
画像・説明・リンクの3点セットだけで、見た目が一気に充実します。

---

## この記事でやること（3行）

- 作品紹介セクションを追加する
- 画像＋説明＋リンクの基本形を作る
- レイアウトを崩さずに整える

## 対象読者

- 1カラム自己紹介ページを作れた人
- スキル一覧を追加できた人

---

## 0. 追加する完成イメージ

- 「Works」見出しの下に作品カード
- 画像＋タイトル＋説明＋リンク
- 1カラムの流れに自然に入る

---

## 1. HTMLにセクションを追加する

`index.html` の `<main class="profile">` 内、スキル一覧の下に追加します。

~~~html
<section class="works">
  <h2>Works</h2>
  <div class="work-cards">
    <article>
      <img src="https://placehold.jp/560x320.png" alt="作品1のサムネイル">
      <h3>作品タイトル1</h3>
      <p>一言で内容を説明します。例：学習記録をまとめるWebサイト。</p>
      <a href="https://example.com" target="_blank" rel="noopener">サイトを見る</a>
    </article>
    <article>
      <img src="https://placehold.jp/560x320.png" alt="作品2のサムネイル">
      <h3>作品タイトル2</h3>
      <p>簡単に特徴を書きます。例：レスポンシブ対応のLP。</p>
      <a href="https://example.com" target="_blank" rel="noopener">サイトを見る</a>
    </article>
  </div>
</section>
~~~

> 画像は仮のダミーです。あとで自分の画像URLに差し替えてOK。

---

## 2. CSSを追加する

`style.css` の最後に追加します。

~~~css
.works {
  margin-top: 32px;
  text-align: left;
}

.works h2 {
  font-size: 18px;
  margin: 0 0 12px;
}

.work-cards {
  display: grid;
  gap: 16px;
}

.work-cards article {
  border: 1px solid #e5e7eb;
  border-radius: 12px;
  padding: 12px;
  background: #fff;
}

.work-cards img {
  width: 100%;
  border-radius: 8px;
  display: block;
}

.work-cards h3 {
  margin: 12px 0 6px;
  font-size: 16px;
}

.work-cards p {
  margin: 0 0 8px;
  font-size: 13px;
  color: #555;
}

.work-cards a {
  color: #1d4ed8;
  text-decoration: none;
  font-weight: 600;
}

.work-cards a:hover {
  text-decoration: underline;
}
~~~

---

## 3. よくあるつまずき

- **画像が大きすぎる**  
  → CSSの `width: 100%` が入っているか確認

- **カードの内側が詰まる**  
  → `padding` を `12px` 以上にすると読みやすい

- **リンクが押しづらい**  
  → テキストは短くし、間隔を空ける

---

## まとめ

- 作品紹介があると「ポートフォリオ感」が一気に出る
- 画像＋説明＋リンクの3点セットが基本
- まずは2作品で十分、増やすのは後からでOK
