**自己紹介ページに「実績リスト」を足して、信頼感をプラスします。**  
資格・受賞・参加イベントなどを、短い箇条書きでまとめます。

---

## この記事でやること（3行）

- 実績リストのセクションを追加する
- 箇条書きで読みやすく整理する
- 見た目を崩さずに整える

## 対象読者

- 1カラム自己紹介ページを作れた人
- スキル/作品紹介/タイムラインを追加できた人

---

## 0. 追加する完成イメージ

- 「Achievements」見出しの下に実績リスト
- 3〜5項目の短い箇条書き
- 1カラムの流れに自然に入る

---

## 1. HTMLにセクションを追加する

`index.html` の `<main class="profile">` 内、タイムラインの下に追加します。

~~~html
<section class="achievements">
  <h2>Achievements</h2>
  <ul class="achievement-list">
    <li>基本情報技術者試験 合格（2024）</li>
    <li>Webデザインコンテスト 参加（2024）</li>
    <li>Techイベント登壇（LT）</li>
    <li>学習記録ブログ 継続中</li>
  </ul>
</section>
~~~

> 中身は「やったこと」を短く書けばOK。

---

## 2. CSSを追加する

`style.css` の最後に追加します。

~~~css
.achievements {
  margin-top: 32px;
  text-align: left;
}

.achievements h2 {
  font-size: 18px;
  margin: 0 0 12px;
}

.achievement-list {
  list-style: disc;
  padding-left: 20px;
  margin: 0;
  color: #222;
  font-size: 14px;
}

.achievement-list li {
  margin-bottom: 8px;
}
~~~

---

## 3. よくあるつまずき

- **箇条書きが左に寄りすぎる**  
  → `padding-left` を `20px` 以上にする

- **行間が詰まる**  
  → `li` の `margin-bottom` を増やす

- **見出しとの距離が近い**  
  → セクションの `margin-top` を増やす

---

## まとめ

- 実績リストがあると「信頼感」が出る
- 箇条書きで十分、長文は不要
- 3〜5項目から始めるのが楽
