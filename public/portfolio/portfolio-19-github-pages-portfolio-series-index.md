各回は短く、手を動かせる内容だけに絞っています。

---

## デモ / リポジトリ（公開後に差し替えOK）

- デモ（GitHub Pages）：`https://<username>.github.io/<repo>/`
- リポジトリ（GitHub）：`https://github.com/<username>/<repo>`

---

## この連載でできるようになること

- HTML/CSSで **1枚の自己紹介ページ** を作れる
- スキル/作品/経歴/実績/連絡先まで **ポートフォリオの定番構成** を揃えられる
- GitHub Pagesで公開し、URLで共有できる
